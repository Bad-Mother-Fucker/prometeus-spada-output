## 5. Proposte

### P-C1-001 — Upgrade tubazioni SN4 → SN8 spiralato composito per tratti su sede stradale provinciale

| Campo | Dettaglio |
|---|---|
| ID | P-C1-001 |
| Gap collegato | G-C1-001 |
| Sub-criterio | C1.1 — Durabilità delle opere (max 20 pt) |
| Descrizione | Sostituzione delle tubazioni PEAD SN4 con tubazioni spiralate composite in acciaio zincato + polietilene (SN8) per tutti i tratti posati in sede stradale provinciale e nelle zone con traffico pesante documentato. La struttura a doppia elica d'acciaio elimina il fenomeno del "creep" sotto carico costante, conferisce rigidità uniforme in ogni sezione, ed è la soluzione esplicitamente identificata dal progettista come migliorativa nel progetto esecutivo (D003 §8.1.VII). I giunti spiralati compositi presentano una rigidità superiore al tubo. |
| Beneficio valutativo | Eliminazione rischio creep su strade provinciali; aumento vita utile ≥30 anni vs standard normativo; riduzione costi manutenzione straordinaria; evidenza diretta nel documento di progetto |
| Compatibilità progetto base | ALTA — stessa geometria di scavo; diametri equivalenti disponibili; stesso metodo di giunzione (bicchiere EPDM). Nessuna variante strutturale |
| Quantificazione | Stimare i metri lineari di rete su sede stradale provinciale dalle planimetrie (D027–D031, D010–D012) |
| Rischi | Costo circa doppio per i tratti interessati; accettabilità economica da valutare rispetto al punteggio attribuibile |

### P-C1-002 — Upgrade calcestruzzi vasche: classe XA1 → XA2 con additivi cristallizzanti

| Campo | Dettaglio |
|---|---|
| ID | P-C1-002 |
| Gap collegato | G-C1-002 |
| Sub-criterio | C1.1 — Durabilità delle opere (max 20 pt) |
| Descrizione | Upgrade della classe di esposizione di tutti i calcestruzzi a contatto con reflui da XA1 a XA2 (moderatamente aggressiva in ambiente solfatico), con aggiunta di additivi cristallizzanti (es. Vandex, Krystol) per impermeabilizzazione in massa del cls. Classe di resistenza minima C30/37 per vasche di ossidazione, denitrificazione, sedimentazione, digestione aerobica. Aggiunta di inibitori di corrosione per armature nei getti delle vasche a contatto diretto con reflui. |
| Beneficio valutativo | Incremento durabilità vasche; riduzione rischio infiltrazione e corrosione; minor manutenzione straordinaria su opere civili; vita utile attesa +20% rispetto a calcestruzzo standard XA1 |
| Compatibilità | ALTA — solo modifica del mix design; nessuna variante geometrica o strutturale |
| Rischi | Costo incrementale moderato (+5–10% per i getti delle vasche); verifica compatibilità con mix design del progettista |

### P-C1-003 — Sistema SCADA integrato: supervisione centralizzata depuratore + 15 stazioni di sollevamento

| Campo | Dettaglio |
|---|---|
| ID | P-C1-003 |
| Gap collegato | G-C1-004 |
| Sub-criterio | C1.4 — Ottimizzazione sistema di controllo (max 8 pt) |
| Descrizione | Installazione di un sistema SCADA (Supervisory Control and Data Acquisition) di nuova generazione che integri in un'unica piattaforma: (1) impianto di depurazione (tutte le utenze: grigliatura, dissabbiatura, comparto biologico, sedimentazione, disinfezione, linea fanghi); (2) 15 stazioni di sollevamento (ST01–ST15); (3) dashboard web-based con KPI in tempo reale. Architettura: upgrade del PLC esistente (Siemens S7 della turbosoffiante) come nodo master; installazione RTU (Remote Terminal Unit) presso ogni stazione di sollevamento con trasmissione dati via GSM/4G; server supervisione in sede o cloud. Funzionalità: allarmi remoti via app mobile, visualizzazione real-time livelli/portate/potenze, reportistica automatica mensile KPI energetici e depurativi, storicizzazione dati per analisi trend. Il telecontrollo esistente sull'impianto viene integrato e potenziato (non sostituito). |
| Beneficio valutativo | Risponde puntualmente a tutti gli elementi premianti di A.4: architettura controllo avanzata; sensoristica intelligente; telecontrollo; gestione remota allarmi; reportistica automatica KPI; resilienza contro guasti; integrazione depuratore-stazioni sollevamento |
| Compatibilità | ALTA — costruisce sul telecontrollo esistente (E-C1-007); compatibile con PLC Siemens S7 già installato |
| Rischi | BASSO — tecnologia consolidata; costo stimabile €80.000–150.000 per full deployment |
